## Phase 1.1: Glassmorphism Design System
**Effort:** Medium | **Priority:** Critical | **File:** app_theme.dart

### Objektif:
Implementasi token-token glassmorphism untuk menciptakan efek kedalaman (depth)
yang khas Apple dengan background blur, border glow, dan multi-layer shadows.

### Perubahan:
1. **Glass Container Tokens**
   - `glassBackground`: Color dengan opacity 0.05-0.15 untuk efek frosted
   - `glassBorder`: Border dengan opacity rendah dan gradient
   - `glassBlur`: BackdropFilter dengan sigma 10-25

2. **Depth/Elevation System**
   - Level 0: Surface (bgCard) - No elevation
   - Level 1: Raised (+8dp shadow + subtle border glow)
   - Level 2: Floating (+16dp shadow + stronger glow)
   - Level 3: Modal (+24dp shadow + ambient light)

3. **Enhanced Shadow System**
   - Inner shadows untuk efek "pressed" state
   - Ambient shadows dengan warna accent
   - Multi-layer shadows (2-3 layers per component)

4. **New Gradient Palette**
   ```dart
   // Premium Glass Gradients
   static const glassGradientPrimary = LinearGradient(
     begin: Alignment.topLeft,
     end: Alignment.bottomRight,
     colors: [
       Color(0x1AB6F09C),  // brandPrimary 10%
       Color(0x0AB6F09C),  // brandPrimary 4%
     ],
   );
   
   // Mesh/Aurora Gradients untuk backgrounds
   static const auroraGradient = LinearGradient(...);
   ```

## Phase 2.1: GlassCard - Premium Card Component
**Effort:** High | **Priority:** Critical | **File:** gym_card.dart â†’ glass_card.dart

### Objektif:
Menggantikan GymCard yang flat dengan GlassCard yang memiliki depth,
blur effect, dan glow borders seperti card di iOS Control Center.

### Implementasi:
```dart
class GlassCard extends StatefulWidget {
  final Widget child;
  final GlassVariant variant; // frosted, solid, gradient, outlined
  final double blurAmount;    // 0-30
  final Color? accentColor;   // untuk border glow
  final bool enablePressEffect;
  final bool enableHoverGlow;
  // ... existing properties
}
```

### Visual Features:
1. **Frosted Glass Effect**
   - `BackdropFilter` dengan blur sigma 15-25
   - Background dengan 5-10% opacity
   - Subtle noise texture overlay

2. **Premium Borders**
   - Top border lebih terang (light source dari atas)
   - Bottom border lebih gelap (3D illusion)
   - Corner radius yang lebih besar (20-24px)

3. **Interactive States**
   - Hover: Border glow dengan accent color
   - Pressed: Scale 0.97 + inner shadow
   - Loading: Shimmer gradient sweep

4. **Variants**
   - `GlassVariant.frosted` - Full blur effect
   - `GlassVariant.solid` - Opaque dengan gradient
   - `GlassVariant.gradient` - Gradient overlay
   - `GlassVariant.outlined` - Border only, transparent bg
